# Bundled characters

The dashboard ships 26 installable bundled fan-art packages: Itachi, Pikachu, Raichu, Charmander, Charizard, Squirtle, Blastoise, Bulbasaur, Venusaur, Eevee, Vaporeon, Jolteon, Flareon, Umbreon, Espeon, Snorlax, Gengar, Psyduck, Meowth, Mew, Mewtwo, Lucario, Dragonite, Piplup, Rowlet, and Cyndaquil.

They are generated by `scripts/generate_bundled_characters.py` from declarative definitions in `codex_usage.pixel_art`. The renderer uses only the standard library, rasterizes 4-frame 1024x256 RGBA sheets, and encodes PNG chunks directly with `struct`, `zlib`, and CRC32. Fixed JSON ordering and fixed ZIP timestamps make repeated builds byte-identical.

Bundled packages appear in the Characters catalog as `bundled`; they are not native/reserved IDs. Selecting **Instalar bundled** validates and installs the package through the existing package registry. Preview sheets load only for the selected character through the existing manifest/asset cache, so the dashboard never downloads all sprite sheets at startup.

Each package has a `NOTICE.txt`: the visual is original simplified non-official fan-art. Character names and associated rights belong to their respective holders; the package does not grant trademark or character rights.
